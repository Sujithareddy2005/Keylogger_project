 ⌨️ Input Logger Web Application (Educational Keylogger)

📌 Project Overview

This project is a **web-based input logging application** built using Flask. It captures user-entered text within the application interface and stores it securely with timestamps.
The goal is to demonstrate how logging mechanisms work in a **controlled and ethical environment**.

 ⚠️ This project is strictly for **educational purposes only** and does NOT capture system-wide keystrokes.

🚀 Features

*  Logs user input from a web form
*  Adds timestamp to each log entry
*  Displays all logged inputs on the screen
*  Clear Logs functionality
*  Simple and clean web interface

🛠️ Technologies Used

* Python
* Flask (Web Framework)
* HTML & CSS

🔐 How It Works

1. User enters text in the input box
2. The application captures the input
3. A timestamp is generated for each entry
4. Data is stored in a local file (`logs.txt`)
5. Logs are displayed on the web page
6. User can clear all logs using a button

▶️ How to Run the Project

1. Install required libraries:
   
pip install flask

2. Run the application:

python app.py

3. Open browser and go to:

http://127.0.0.1:5000/
